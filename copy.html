What is Visual Coding?

Visual programming language uses blocks or graphics that are pre-defined with code to create programs. You can build interactive stories, games, and animations by connecting these blocks!

Key Components

Explore the main components of Visual Coding: Sprite, Stage, and Script!

Sprite

A Sprite is a character or object in Visual Coding. It is the element that performs actions on the stage. You can choose, customize, or upload your own sprite to make your programs unique.

Stage

The Stage is the background where all the actions happen. It is the canvas where your sprites interact and perform the coded actions.

Script

The Script is the set of instructions or code blocks you arrange to make your sprites perform actions. It defines the behavior of your sprites and brings your story, game, or animation to life.
