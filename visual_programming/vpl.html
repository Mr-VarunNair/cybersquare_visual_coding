What is Visual Coding?

Visual programming language uses blocks or graphics that are pre-defined with code to create programs. You can build interactive stories, games, and animations by connecting these blocks!

Key Components

Explore the main components of Visual Coding: Sprite, Stage, and Script!

Sprite

A Sprite is a character or object in Visual Coding. It is the element that performs actions on the stage. You can choose, customize, or upload your own sprite to make your programs unique.

Stage

The Stage is the background where all the actions happen. It is the canvas where your sprites interact and perform the coded actions.

Script

The Script is the set of instructions or code blocks you arrange to make your sprites perform actions. It defines the behavior of your sprites and brings your story, game, or animation to life.

Scripting Area

The Scripting Area is the large empty space to the right of the Block Palette. You can drag the programming blocks and create programs for each sprite in the scripting area.

Sprite Pane

Sprite pane is a white area located just below the Stage. It contains options to change the size, name, position and direction of Sprite.

Blocks Palette

Code blocks are broadly classified into 9 categories which are colour-coded. Each category has a set of programming blocks under it. We can use these programming blocks to make the sprite act according to our needs.

Interactive Activities


🐾 Hello! 🐱

Let’s make the Meow say Hello!
Follow these easy steps to unleash your creativity:

  1. Open the Visual Coding environment.
  2. Drag the "When Flag Clicked" block from the Events category.
  3. Drag the "Say Hello! for 10 seconds" block from the Looks category.
  4. Click the green flag and watch your sprite say Hello!

Tip: You can change the message and seconds.


🐾 Talking Cats 🐱

Come, let us learn how to make a conversation between sprites! Follow these fun steps to create your animation:

  1. Open the Visual Coding environment.
  2. Add one more sprite by clicking "Choose a Sprite"option.
  3. Select the first sprite.
  4. Drag the "When Flag Clicked" block from the Events category.
  5. Drag the "Say Hello! for 10 seconds" block from the Looks category.
  6. Now select the second sprite.
  7. Drag the "When Flag Clicked" block from the Events category.
  8. Drag the "Say Hello! for 10 seconds" block from the Looks category.
  9. Click the green flag and watch both cats say Hello!

Tip: Experiment with different sprites, messages and time.


🖊️ Draw Lines ✏️

Let’s make your sprite draw lines on the stage! Follow these easy steps to unleash your creativity:

  1. Open the Visual Coding environment and choose a sprite.
  2. Click on "Add Extension" at the bottom left and select the Pen extension.
  3. Drag the "When Flag Clicked" block from the Events category.
  4. Add the "Erase all"block from the Pen category.
  5. Add the "Set pen color to: "block from Pen category and change color to Red.
  6. Drag the "Move 50 steps" block from the Motion category.
  7. Drag the "Pen down" block from the Pen category.
  8. Repeat the 6th and 7th steps 3 times.
  9. Drag the "Move 100 steps" block from the Motion category.
  10. Click the green flag and watch your sprite draw a line!

Tip: You can change the sprite, color of line and size of line!



🖊️ Draw a Square ✏️

Let’s make your sprite draw a Square on the stage! Follow these easy steps to unleash your creativity:

  1. Open the Visual Coding environment and choose a Pen ✏️ sprite.
  2. Click on "Add Extension" at the bottom left and select the Pen extension.
  3. Drag the "When Flag Clicked" block from the Events category.
  4. Add the "Erase all"and "Pen up" block from the Pen category.
  5. Drag the "go to x : 0 y : 0" block from the Motion category.
  6. Add the "Set pen color to: "block from Pen category and change color to Red.
  7. Add the "Set pen size to: " block from the Pen category and change size to 5.
  8. Drag the "Pen down" block from the Pen category.
  9. Drag the "Repeat" block from the Control category and set to 4.
  10. Drag the "Move 100 steps"and "Turn 90 degrees" block from the Motion category and place inside the repeat block.
  11. Click the green flag and watch your sprite draw a line!

Tip: You can change the sprite, color of line and size of line!
