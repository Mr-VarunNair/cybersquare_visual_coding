What is Visual Coding?

Visual programming language uses blocks or graphics that are pre-defined with code to create programs. You can build interactive stories, games, and animations by connecting these blocks!

Key Components

  • Sprite: A moving object in Visual Coding, like a cat. You can choose, paint, or upload your own sprite!
  • Scripting Area: Drag and drop code blocks here to create programs for your sprite.
  • Stage: The area where the action happens!
  • Blocks Palette: Contains all the coding blocks categorized by color.
